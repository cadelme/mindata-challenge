# 🚀 Mindata Challenge - Angular 21

Este proyecto es la resolución de la prueba técnica para **Mindata**. Se ha desarrollado una interfaz moderna, robusta y accesible utilizando **Angular 21** y **Tailwind CSS**, con un enfoque "Pixel Perfect" basado en las especificaciones de diseño.

## 🛠️ Stack Tecnológico
* **Framework:** [Angular 21.2.3](https://angular.dev/)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
* **Tipografía:** Inter & Figtree (Google Fonts)
* **Testing:** Vitest
* **Iconografía:** SVG Assets optimizados

## 📦 Instalación y Uso

1.  **Clonar el repositorio:**
    ```bash
    git clone [https://github.com/cadelme/mindata-challenge.git](https://github.com/cadelme/mindata-challenge.git)
    ```
2.  **Instalar dependencias:**
    ```bash
    npm install
    ```
3.  **Ejecutar en desarrollo:**
    ```bash
    ng serve
    ```
    Navega a `http://localhost:4200/`.

4.  **Ejecutar Tests:**
    ```bash
    ng test
    ```
